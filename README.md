CHECKERS
========

Simple checkers game written in common lisp.
Uses lispbuilder to get the input, do rendering and other stuff.

#### Keys
* q - quit
* left, up, right, down - move selection
* space - initialize, apply or cancel move depending on context
* escape - cancel move
